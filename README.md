# Holiday API

This API allows you to list and confirm holidays. It is built using Java 17, Spring Boot 3.2.5, and connects to an Oracle database.

## Features

- List all holidays
- Confirm a specific holiday

## Prerequisites

- Java 17
- Maven 3.6+
- Oracle Database
